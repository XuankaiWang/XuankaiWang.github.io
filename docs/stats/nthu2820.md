# NTHU MATH2820: Statistics

## 课程简介

- 所属大学：NTHU
- 先修要求：微积分, 线性代数, 概率论
- 课程难度：🌟🌟🌟
- 预计学时：

郑少伟老师的数理统计课程, 视频开二倍速直接刷下来, 讲得十分清楚, 会讲到自己通俗易懂的理解方式

与 MIT 18.650 用的是相同的教材, 国立清华大学的这个课程会先把概率论给过一遍, 而 MIT 18.650 直接从三大统计分布开始讲起, 因此可以根据自己的学习进度, 结合两个课程一起来学习.
